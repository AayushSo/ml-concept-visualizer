Dimensions

W (Width): width of the input image (pixels per row).

K (Kernel Size): width of the convolution filter. Here, K = 3 for a 3×3 kernel.

Formula (stride = 1, padding = 0): Output Width = W - K + 1

In this demo: W = 10, K = 3 → Output Width = 8.
